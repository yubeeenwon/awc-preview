Key UI — 사이트 핵심 아이콘

실제 웹사이트에서 가장 자주 보이는 아이콘. 다른 방에서 적용할 때 이 SVG를 그대로 복사.

hamburger 모바일 내비 토글
hamburger-close 메뉴 닫기 (X 전환)
chatbot 채팅 FAB 버튼
send-message 채팅 입력 전송
cta-arrow Contact · View all 버튼
modal-mail 문의 모달 아이콘
page-back 서브페이지 ← Back

/* 햄버거: CSS span 3줄 → SVG 교체 */
/* 기존: .nav-burger span × 2 (CSS 선) */
/* 신규: 중앙 갭 있는 SVG 2줄 */

/* 챗봇 FAB: ico-chat 교체 */

  <path d="M3 21v-4"/>
  <path d="M3 13V5a2 2 0 0 1 2-2h14a2 2 0 0 1 2 2v10"/>
</svg>

/* 챗봇 X: ico-x 교체 */
<svg class="ico-x" viewBox="0 0 24 24">
  <path d="M18 6l-5.5 5.5"/>
  <path d="M6 18l5.5-5.5"/>
  <path d="M6 6l5.5 5.5"/>

</svg>

</svg>
